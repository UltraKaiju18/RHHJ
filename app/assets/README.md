# Assets

Place static assets used by the RHHJ mobile application in this folder.

## Sub-folders

| Folder | Description |
|--------|-------------|
| `images/` | Logos, background images, and illustrations |
| `icons/` | App icon and custom icons |
| `fonts/` | Custom font files (`.ttf` / `.otf`) |

## Asset Guidelines

- **Images** — use PNG for logos/icons (transparent background); JPEG for photos.
- **Resolution** — provide at minimum a 2× resolution image for high-DPI screens.
- **Naming** — use lowercase, hyphen-separated names (e.g., `rhhj-logo.png`, `home-visit-icon.svg`).
