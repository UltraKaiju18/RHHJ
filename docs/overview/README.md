## Program Summary

**Rays of Hope Hospice Jinja (RHHJ)** provides home-based palliative and curative care for cervical cancer patients in Uganda. The digital health solution being developed in this repository aims to:

1. Replace paper-based and Excel data collection with a structured mobile application.
2. Enable care workers to capture patient data offline on handheld devices.
3. Synchronise all data with a central cloud-hosted SQL database (Supabase) when connectivity is available.
4. Provide programme managers with real-time dashboards and reporting.
