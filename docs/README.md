# Documentation

This folder contains all project documentation for the RHHJ digital health solution.

## Sub-folders

| Folder | Purpose |
|--------|---------|
| [`overview/`](overview/) | Program background, scope, objectives, and stakeholder information |
| [`process-diagrams/`](process-diagrams/) | Workflow diagrams, patient journey maps, and process flow charts |
| [`data-structures/`](data-structures/) | Existing Excel data collection forms and data dictionaries used for database design |

## Adding Documents

- **Overview documents** — Place PDFs, Word documents, or Markdown files in `overview/`.
- **Diagrams** — Place PNG, SVG, JPEG, or Visio/Lucidchart exports in `process-diagrams/`. Use descriptive file names (e.g., `patient-intake-flow.png`).
- **Data structures** — Place Excel workbooks or CSV exports in `data-structures/`. These will inform the SQL schema design in [`../database/schemas/`](../database/schemas/).
